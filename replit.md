# saloniiiiii_dsa

A Python DSA (Data Structures and Algorithms) practice repository.

## Project Overview

Collection of standalone Python scripts demonstrating fundamental data structures and algorithms concepts.

## Scripts

- `stack.py` - Stack operations (LIFO) using a Python list
- `queue.py` - Queue operations (FIFO) using a Python list
- `tuple.py` - Basic tuple operations and indexing
- `tuple2.py` - Advanced tuple operations (counting, indexing, slicing, unpacking)
- `dictionary1` - Nested dictionary example (Library Management system)
- `dictionary2.py` - Basic dictionary operations
- `main.py` - Runner script that executes all scripts in sequence

## Running

The workflow runs `python3 main.py` which executes all DSA scripts in sequence and prints their output to the console.

## Environment

- Language: Python 3.12
- No external dependencies (uses only Python built-ins)
- No frontend or backend server
